# \<net-viel-thread>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i net-viel-thread
```

## Usage
```html
<script type="module">
  import 'net-viel-thread/net-viel-thread.js';
</script>

<net-viel-thread></net-viel-thread>
```
