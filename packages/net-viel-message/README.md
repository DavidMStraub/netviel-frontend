# \<net-viel-message>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i net-viel-message
```

## Usage
```html
<script type="module">
  import 'net-viel-message/net-viel-message.js';
</script>

<net-viel-message></net-viel-message>
```
